# Access Control

## Режим работы

Приложение переведено в **single-user** режим:

- Один оператор работает локально в одном интерфейсе.
- Комнаты, онлайн-роли и распределение прав между участниками не используются.
- Все функции (карта, цели, расчёты, корректировки, журналы) доступны одному пользователю.

## Примечания

- Валидация данных и журналирование операций сохраняются.
- Многопользовательская RBAC-модель в текущей версии отключена как избыточная.
