# shared-types

Минимальные контракты API для синхронизации приложений и сервисов.

## Контракты

- `GunType` — описание орудия и его характеристик.
- `Target` — цель и её параметры.
- `SalvoRequest` — запрос на залп.
- `Correction` — корректировка от наблюдателя.
- `CrewStatus` — статус расчёта/орудия.

См. реализацию контрактов в `index.ts`.
