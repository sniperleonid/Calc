# web-map

Модуль веб-карты теперь построен как **тактическое рабочее пространство** с изолированными подсистемами:

- `ballistics` — данные орудий и расчётных позиций;
- `observers` — наблюдатели и БПЛА;
- `logistics` — снабжение и запросы;
- `infantry` — позиции пехоты;
- `map` — загрузка карты, калибровка, метки и инструменты.

## Основные сценарии

- Ручной ввод данных в модули (`upsertManual`).
- Размещение сущностей прямо на карте (`placeOnMap`).
- Загрузка собственного изображения карты (`loadMap`).
- Сохранение/загрузка калибровки (`saveCalibrationPreset`, `loadCalibrationPreset`).
- Сохранение/загрузка миссии целиком (`saveMission`, `loadMission`).
- Формирование схемы интерфейса для интуитивных панелей ввода (`buildUiSchema`).

Реализация: `src/tactical-workspace.js`.
